# CatPhish Icons

Place your icon files here:
- `icon16.png`   — 16×16 px  (favicon / toolbar)
- `icon48.png`   — 48×48 px  (extensions page)
- `icon128.png`  — 128×128 px (Chrome Web Store)

## Quick Start (for hackathon)

You can use any 🦊 fox emoji icon or generate simple PNG icons.
A fast free option: https://favicon.io/emoji-favicons/fox/

Download the ZIP and rename the files to `icon16.png`, `icon48.png`, `icon128.png`.

The extension will still load without icons — Chrome will just show a default puzzle piece.
